# AutoReady
Automatically skips the ready screen when loaded

### Installation
- Install [BepInEx](https://v-rising.thunderstore.io/package/BepInEx/BepInExPack_V_Rising/)
- Extract _AutoReady.dll_ into _(VRising folder)/BepInEx/plugins_
